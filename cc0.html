cc0 notice

effective date: jan 1, 2026

mediaBrilliance digitalxtudio sometimes releases specific materials under CC0 1.0 Universal (a public domain dedication). when something is marked “cc0,” you may copy, modify, distribute, and use it for any purpose — including commercial use — without asking permission.

what this applies to

cc0 applies only to items where we explicitly say it applies (for example: a download, a repository, a file header, or a page note that says “cc0”).

what this does not apply to

  • client work or commissioned deliverables, unless explicitly marked cc0
  • third-party assets (fonts, icons, photos, libraries) that have their own licenses
  • trademarks, brand names, or logos, except where explicitly released under cc0
  • privacy/publicity rights (people’s names, likenesses, or personal data)

official cc0 text

for the complete, authoritative license text, use the official creative commons pages:

  • cc0 1.0 universal deed (human-readable summary)
  • cc0 1.0 universal legal code (full text)

questions

if you’re not sure whether something is cc0, reach out and we’ll clarify: mediabrilliance.io/contact.html

this notice is provided for clarity and convenience. the official cc0 legal code linked above controls in case of any difference.
