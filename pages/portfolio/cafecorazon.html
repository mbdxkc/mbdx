overview

cafe corazon is a beloved latin/indigenous coffee shop seeking to strengthen their digital presence and improve local search visibility. the project focused on comprehensive seo optimization and website performance improvements.

seo strategy

local-first approach. implemented coffee shop schema markup for rich search results, optimized google business profile with accurate contact details, and built local citation consistency across directories.

technical foundation. improved core web vitals through image compression, lazy loading, and resource optimization. created xml sitemap, robots.txt, and canonical url structure for clean indexing.
