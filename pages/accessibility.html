accessibility statement

This statement was last updated on January 1, 2026.

We at mediaBrilliance are committed to ensuring that our website, mediabrilliance.io, is accessible to people with disabilities. We are actively working to improve the user experience for everyone, and we aim to comply with relevant accessibility standards, including the Web Content Accessibility Guidelines (WCAG) 2.1.

If you experience any difficulty accessing content on this site, or if you have suggestions for how we can improve accessibility, please contact us via our contact page. We value your feedback and will do our best to accommodate your needs.

what web accessibility is

An accessible site allows visitors with disabilities to browse the site with the same or a similar level of ease and enjoyment as other visitors. This can be achieved with the capabilities of the system on which the site is operating, and through assistive technologies.

accessibility adjustments on this site

We have adapted this site in accordance with WCAG 2.1 guidelines and have made the site accessible to the level of AA. The site’s content is designed to work with assistive technologies such as screen readers and keyboard navigation.

measures we have implemented

  • Aligned the site with WCAG 2.1 Level AA standards
  • Ensured compatibility with assistive technologies (e.g., screen readers, keyboard navigation)
  • Defined a clear and consistent heading structure
  • Set the site’s default language for assistive software
  • Organized content in a logical reading order
  • Added descriptive alternative text to essential images
  • Used high-contrast, accessible color combinations
  • Minimized the use of motion and animations
  • Worked to make videos, audio files, and downloadable documents accessible when present
